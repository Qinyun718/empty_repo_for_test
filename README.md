# CRM Dashboard Example

This repository now contains a simple HTML/JavaScript CRM dashboard. To view it, open `src/index.html` in your browser. The page allows you to add contacts and shows a running total.

## Files

- `src/index.html` – main dashboard page
- `src/style.css` – basic styling
- `src/script.js` – front-end logic for managing contacts

No additional dependencies or build steps are required.
